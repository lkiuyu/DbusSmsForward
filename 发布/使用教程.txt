1.输入
sudo apt install libicu67
安装libicu

2.解压程序至home文件夹下

3.输入 
sudo chmod -R 777 DbusSmsFoward
配置程序可执行权限

4.输入
./DbusSmsFoward
运行程序

5.根据体术配置相关邮箱信息

6.带参数运行跳过程序初始的运行模式选择以达到快速运行程序

输入
./DbusSmsForward -fE
跳过运行模式选择直接进入邮箱转发模式

输入
./DbusSmsForward -fP
跳过运行模式选择直接进入PushPlus转发模式

输入
./DbusSmsForward -fW
跳过运行模式选择直接进入企业微信转发模式

输入
./DbusSmsForward -sS
跳过运行模式选择直接进入短信发送界面
